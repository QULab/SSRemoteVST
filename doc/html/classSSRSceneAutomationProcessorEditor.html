void SSRSceneAutomationProcessorEditor::timerCallback ( )
overridevirtual

Inherited from the JUCE Timer Class.

Like the JUCE documentation say: "The user-defined callback routine that actually gets called periodically."

Currently, this method calls the processors method to read the incoming messages and then sets all the UI elements to the new values held by the processor if there have been any changes.
