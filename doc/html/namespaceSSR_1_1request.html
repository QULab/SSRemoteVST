void SSR::request::source_fixed ( std::ostream *  output,
const int  source_id,
const bool  fixed

This function writes the source fixed XML request statement into output as follows:

<source id="source_id"><position fixed="fixed"/></source>

For more detailed information please visit: http://ssr.readthedocs.org

Parameters

source_idThe source ID.
fixedTrue if the source shall be fixed
