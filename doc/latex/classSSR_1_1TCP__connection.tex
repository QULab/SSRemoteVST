\subsection{Detailed Description}
This class represents a tcp client connection and is basically a wrapper class of the J\-U\-C\-E class Streaming\-Socket.

This class provides methods to connect to a host and disconnect from it, methods to get messages from the host and send messages to the host and a method to check whether this client is connected to the host.

Warning\-: This class should implement an interface allready written but this does not work properly, so this is something to work on...
